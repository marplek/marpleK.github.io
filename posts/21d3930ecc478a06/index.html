判斷點是否在多邊形內

Posted by MarpleK on Sat, Mar 5, 2022

射線法就是以判斷點開始，向右（或向左）的水平方向作一射線，計算該射線與多邊形每條邊的交點個數，如果交點個數為奇數，則點位於多邊形內，偶數則在多邊形外。該算法對於復合多邊形也能正確判斷。

射線法的關鍵是正確計算射線與每條邊是否相交。並且規定線段與射線重疊或者射線經過線段下端點屬於不相交。首先排除掉不相交的情況，下圖的情況都是需要排除掉的：

以c++實現

 1#include <iostream>
 2#include <vector>
 3
 4struct Point {
 5    double x;
 6    double y;
 7};
 8
 9bool point_in_polygon(Point point, std::vector<Point> polygon) {
10    int polygon_length = polygon.size();
11    int j = polygon_length - 1;
12    bool is_in_polygon = false;
13
14    for (int i = 0; i < polygon_length; i++) {
15        if (((polygon[i].y < point.y && polygon[j].y >= point.y) || (polygon[j].y < point.y && polygon[i].y >= point.y)) &&
16            (polygon[i].x <= point.x || polygon[j].x <= point.x)) {
17            if (polygon[i].x + (point.y - polygon[i].y) / (polygon[j].y - polygon[i].y) * (polygon[j].x - polygon[i].x) < point.x) {
18                is_in_polygon = !is_in_polygon;
19            }
20        }
21        j = i;
22    }
23
24    return is_in_polygon;
25}

 1def point_in_polygon(point, polygon):
 2    polygon_length = len(polygon)

 4    is_in_polygon = False
 5
 6    for i in range(polygon_length):
 7        if ((polygon[i][1] < point[1] and polygon[j][1] >= point[1]) or
 8            (polygon[j][1] < point[1] and polygon[i][1] >= point[1])) and \
 9                (polygon[i][0] <= point[0] or polygon[j][0] <= point[0]):
10            if polygon[i][0] + (point[1] - polygon[i][1]) / (polygon[j][1] - polygon[i][1]) * (polygon[j][0] - polygon[i][0]) < point[0]:
11                is_in_polygon = not is_in_polygon
